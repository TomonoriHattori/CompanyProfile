Q&A

Is the ailia SDK paid?

The ailia SDK is a paid library, and a license agreement with our company is required for commercial use.

Can the ailia SDK be evaluated for free?

The evaluation of the ailia SDK can be done for free. The evaluation version can be obtained from our WEB site or via python's pip.

Is there a cost for individuals using the ailia SDK for non-commercial purposes?

If an individual uses it for non-commercial purposes or small-scale commercial purposes, it can be used for free.
Small-scale commercial purposes refer to cases where the total economic benefit (sales revenue, advertising revenue, and other economic benefits) from the user's product over 12 months is less than $100,000.
When using the application, it is necessary to specify that ailia SDK was used and to clearly indicate the link to ailia SDK (Article 2, Section 3).
The use of this software in commissioned development by third parties is not intended for personal use.

Is there a cost for businesses using the ailia SDK for non-commercial purposes?

Even for non-commercial purposes, a license agreement with our company is required if businesses distribute the app they created.

Is there a cost for using the ailia SDK for academic purposes?

The ailia SDK can be used for free for individual academic purposes (such as writing a paper).
Additionally, it is necessary to state that the ailia SDK was used and provide a link to the ailia SDK when used in a paper.
In the case of joint research with businesses, a separate agreement is necessary.

Are there any restrictions on evaluating the ailia SDK?

There are no functional restrictions.

Is there a time limit for evaluating the ailia SDK?

A license file is required for evaluating the ailia SDK, and it will not be available after the 30-day license evaluation period has expired..
If you obtained the SDK from the WEB site, you can extend the evaluation period by requesting the evaluation version again..
If obtained via python's pip, the license file will be automatically downloaded every 30 days.

Can I write about how to use the ailia SDK on a BLOG, etc.?

No problem. We hope you can share information about the ailia SDK.

Can I reference ailia SDK from OSS?

It's also no problem to include ailia in the OSS requirements.txt.
